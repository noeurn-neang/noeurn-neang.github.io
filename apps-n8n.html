n8n Google API Automation Project

About This Project

n8n Google API Automation is a personal project by Noeurn Neang, a Cambodian software engineer with over 10 years of experience in web development and automation. It leverages n8n to integrate with Google APIs, enabling seamless automation for data processing, file management, and scheduling tasks.

Project Functionality

This project automates workflows using Google APIs, including:

  • Google Sheets: Read/write data for analysis, reporting, or syncing (e.g., sales trends, inventory updates).
  • Google Drive: Manage files/folders for storage or automated sharing.
  • Google Calendar: Schedule events or reminders based on workflow triggers.
  • Other Google APIs: Integrate APIs like Gmail for email automation or Google Data Studio for visualizations.
  • Custom n8n workflows for personal, business, or open-source use.

Data Usage and Transparency

This project requests access to Google APIs (e.g., Sheets, Drive, Calendar) only to execute your configured n8n workflows. Access is limited to resources you explicitly authorize via Google OAuth (e.g., specific spreadsheets, files, or calendars). Data is processed in real-time and not stored on external servers beyond workflow execution. No additional personal information is collected or shared. For details, see our Privacy Policy.
